# Claude Code Skills (by Arvind Raman)


## Installation

Clone the repo locally, then install as a Claude Code plugin:

```bash
git clone https://github.com/arvindraman/skills.git ~/.claude/plugins/skills
```

Then in Claude Code:

```
/plugin install ~/.claude/plugins/skills
```

## Skills

| Skill | Command | When to use |
|-------|---------|-------------|
| **agentforce-prioritization** | `/agentforce-usecase` | Use this skill whenever a user wants to evaluate, score, or prioritize Agentforce use cases for an MVP implementation |
